# Data Explorer

Data Explorer is a small app project for exploring uploaded datasets.

## Development Flow

This repo uses a branch-and-pull-request workflow for every implementation.

1. Create or choose one issue.
2. Create a feature branch from `main`.
3. Implement only that issue's scope.
4. Run the available checks.
5. Open a pull request back to `main`.
6. Review, update, and merge.

Branch names should be short and issue-focused:

```text
feature/1-scaffold-app
feature/2-csv-upload
fix/7-empty-file-error
```
